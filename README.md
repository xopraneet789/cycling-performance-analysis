# Cycling Performance Analysis

This repository contains Python scripts used for the statistical analysis of rider performance across different stage types in a simulated cycling dataset.

## Files
- `cycling.txt` : Dataset used for the analysis
- `stats.py` : Statistical tests and model fitting (ANOVA, Kruskalâ€“Wallis)
- `tables.py` : Generation of summary tables used in the report

## Analysis Overview
The analysis investigates whether rider class and stage class influence performance scores and whether interaction effects exist between these factors.
